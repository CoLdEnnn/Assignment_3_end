# Assignment_3_end

### Blog API
A full-stack web application featuring a RESTful API built with Node.js and Express, integrated with MongoDB Atlas, and a reactive frontend inspired by SwiftUI design principles.

### Features
- CRUD Operations: Full Create, Read, Update, and Delete functionality for blog posts.
- Database Integration: Powered by MongoDB Atlas with Mongoose schemas.
- RESTful API: Clean endpoints for managing data.
- Reactive Frontend: A user-friendly interface that interacts with the API in real-time.
- Environment Safety: Secure configuration using .env for database credentials.
- Cross-Platform: Built using path module and cors to work across different OS and environments.
